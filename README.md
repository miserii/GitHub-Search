# GitHub-Search
[![Language](https://img.shields.io/badge/language-Swift%205.3.2-orange.svg)](https://swift.org)
[![Xcode](https://img.shields.io/badge/Xcode-12.2-blue.svg)](https://developer.apple.com/xcode)

GitHubAPIを使用したアプリ
- リポジトリ一覧
- インクリメンタルサーチ
- 検索結果のスター順、作成日順での並び替え
- WebViewで各リポジトリに遷移
- Cellを右スワイプしてブックマーク
- [WIP]ViewModelのテスト
を実装
